  1. Jelaskan apa itu PHP
    1. Pengertian

      2. PHP (Hypertext Preprocessor) adalah bahasa pemrograman open-source yang umumnya digunakan untuk membangun aplikasi web dinamis dan intraktif. Fungsinya berkaitan dengan kinerja server dan kelancaran sebuah website. Bahasa PHP sendiri tersedia pada beberapa sistem operasi seperti linux, windows, macOS. Kita dapat memungkinkan untuk menyisipkan kode PHP di dalam dokumen HTML dan dapat menghasilkan halaman wab yang dihasilkan secara dinamis.

    2. Sebutkan keunggulan PHP dibandingkan dengan bahasa pemrograman lainnya dalam konteks pemgembangan web
      • Mudah Dipelajari : PHP tidak terlalu sulit untuk di pelajari dari bahasa lainnya, dan dapat dilakukan modifikasi beberapa bagian HTML dengan mudah oleh pemula. Dan PHP tidak jauh berbeda dengan HTML yang relatif sederhana.
      • Open Source : sumber dari kode dapat diakses oleh siapa pun secara gratis, dan para pengembang dapat memodifikasi sesuai dengan kebutuhan.
      • Kemudahan Integrasi : PHP terhubung dengan berbagai jenis basis data seperti MySQL, PostgreSQL, SQLite, dan semacanya. Selain itu PHP dapat dijalankan dari berbagai OS Komputer seperti Linux, Windows, dan mecOS. Serta PHP memeiliki penyedia layanan web server yang bisa digunakan yaitu, Apache, Nginx, dan IIS.
      • Tingkat Keamanan yang Tinggi : dalam hal ini user tidak dapat melihat penulisan dari kode web secara langsung dikarnakan file PHP terismpan didalam serer database. Terdapat pula teknik hashing yang digunakan untuk menyamarkan password dalam basis data.

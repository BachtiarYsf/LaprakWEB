  1. Penggunaan include, require, dan koneksi PHP dengan MySQL
    1. Jelaskan fungsi include dan require dalam PHP, termasuk perbedaan mendasar antara keduanya.
      • Include : metode yang akan tetap mengeksekusi baris kode selanjutnya ketika file yang diload ternyata tidak ada.
      • Require : metode ini tidak akan melanjutkan pengeksekusian kode program berikutnya ketika file yang di load ternyata tidak ada.
      • Connect : metode ini tidak akan melanjutkan pengeksekusian kode program berikutnya ketika file yang di load ternyata tidak ada.
      • Perbedaan
        1. Include : Akan memunculkan peringatan (E_WARNING) dan skrip akan dilanjutkan.
        2. Require : Akan memunculkan kesalahan fatal (E_COMPILE_ERROR) dan menghentikan skrip.
        3. Connect : mengubah ekstensi file PHP eksternal dan menghubungkannya ke HTML.

  1. Perbandingan GET dan POST
    1. Jelaskan perbedaan antara metode GET dan POST dalam penggunaan disebuah website.
      • GET : digunakan untuk mendapatkan data dari server tanpa harus mengirimkan data tambahan. Ini berarti ketika kita menggunakan tautan untuk emmbuka sebuah halaman, metode get akan digunakan secara otomatis, karena kita tidak mengirimkan informasi tambahan ke server.
      • POST : digunakan untuk mengirim data ke server secara tersembunyi. Hal ini memastikan agar data yang dikirmkan aman dan tidak tersesat ketika dikirimkan ke server.
    2. Berikan contoh kasus penggunaan yang tepat untuk masing masing metode.
      • GET : dikarnakan data yang dikirmkan akan ditampilkan di halaman web, penggunaan GET sangat cocok untuk informasi yang tidak sensitif seperti misalnya infromasi terkait berita viral.
      • POST : penggunaannya ketika kita ingin mengirinkan data dari form suatu website ke server. Data yang dikirmkan menggunakan metode ost sangat terenkripsi dan tidak bisa ditangkap oleh pengguna lain. Metode ini digunakan untuk transaksi e-commerce atau pengiriman data sensitif lainnya
